Afin d'ajouter XITI en pied de page de votre thème, le thème doit comporter le code {{tpl:SysBehavior behavior="publicFooterContent"}}, généralement dans le fichier de thème _footer.html.

Vous pouvez utiliser à la place le widget XITI mais ce plugin doit être activé et bien configuré dans la présente page.

Dans le pied de page du template, XITI est entouré d'une balise "div" de classe "xiti-footer".

Dans le widget, XITI est entouré d'une balise "div" de classe "xiti-widget".
