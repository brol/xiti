In order to add XITI to your theme footer, theme must have sysBehavoir publicAfterContent, commonly in template file _footer.html.

You can use instead the XITI widget but this plugin must be enabled and well configured here.

In footer template, XITI is encapsuled in a "div" tag of class "xiti-footer".

In widget, XITI is encapsuled in a "div" tag of class "xiti-widget".
